# Mono Repo Aplikasi CUTI

Adalah mono repo untuk aplikasi sederhana pengelolaan data cuti. Backend menggunakan `nest.js` dan frontend menggunakan `next.js` dengan database `mariadb`.


## Installasi

Clone satu repo ini.

Untuk backend

```bash
cd api
npm install
npm run dev
```

Untuk frontend

```bash
cd frontend
npm install
npm run dev
```
